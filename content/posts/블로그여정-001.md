# 1. Markdown 및 관련 기본 설정
archetypes -> default.md 설정을 통해, 글쓰기를 시작할때
.md 파일에 기본적으로 템플릿이 완성되는 구조로 작성된다.

# 2. 기본적인 Markdown 생성 방법
```hugo new posts/[생성방법].md```
이런식으로 작성할 경우 마크다운이 생성되면서 기본적인 템플릿이 나온다.
